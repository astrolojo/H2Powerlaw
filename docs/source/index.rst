H2Powerlaw
###########

``H2Powerlaw`` is a Python implementation of the H2 excitation model of `Togi & Smith (2016) <https://ui.adsabs.harvard.edu/abs/2016ApJ...830...18T/abstract>`_, which assumes a continuous, power-law distribution of H2 temperatures.

Installation
===============

``H2Powerlaw`` can be installed using pip, either directly from the git repo::

    # from the master trunk on the repository
    pip install git+https://github.com/astrolojo/H2Powerlaw

...or through PyPI (coming soon!)

Reporting Issues
==================

If you have found a bug in ``H2Powerlaw``, please report it by creating a
new issue on its `GitHub issue tracker
<https://github.com/astrolojo/H2Powerlaw/issues>`_.

Please include an example that demonstrates the issue sufficiently so that the
developer can reproduce and fix the problem.
